# <a name="wdsutil-remove"></a>WDSUTIL entfernen

> Gilt für: Windows Server (halbjährlicher Kanal), Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

## <a name="subcommands"></a>Unterbefehle
|Subommand|BESCHREIBUNG|
|-------|--------|
|[Befehl "Remove-Image" mit WDSUTIL](wdsutil-remove-image.md)|entfernt ein Start-oder Installations Abbild vom Server.|
|[WDSUTIL Remove-ImageGroup-Befehl](wdsutil-remove-imagegroup.md)|entfernt eine Abbild Gruppe vom Server.|
|[WDSUTIL Remove-MulticastTransmission-Befehl](wdsutil-remove-multicasttransmission.md)|Deaktiviert die Multicast Übertragung eines Bilds.|
|[WDSUTIL Remove-Namespace-Befehl](wdsutil-remove-namespace.md)|entfernt einen Namespace vom Server.|
|[WDSUTIL Remove-drivergrouppackage-Befehl](wdsutil-remove-drivergrouppackage.md)|entfernt ein Treiber Paket aus einer Treiber Gruppe auf einem Server.|
|[WDSUTIL Remove-drivergrouppackages-Befehl](wdsutil-remove-drivergrouppackages.md)|entfernt Treiber Pakete aus einer Treiber Gruppe auf einem Server.|
|[WDSUTIL Remove-DriverPackage-Befehl](wdsutil-remove-driverpackage.md)|entfernt ein Treiber Paket von einem Server.|
|[WDSUTIL Remove-driverpackages-Befehl](wdsutil-remove-driverpackages.md)|entfernt Treiber Pakete von einem Server.|
|[WDSUTIL Remove-drivergroup-Befehl](wdsutil-remove-drivergroup.md)|entfernt eine Treiber Gruppe von einem Server.|
|[WDSUTIL Remove-drivergroupfilter-Befehl](wdsutil-remove-drivergroupfilter.md)|entfernt eine Filterregel aus einer Treiber Gruppe auf einem Server.|
